### Manual Installation
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The extension will be installed and ready to use!

## Usage

1. **Pomodoro Timer**: Set work/break times and start focusing
2. **Task Management**: Add tasks with priorities and due dates
3. **Analytics**: View your focus time and productivity stats
4. **Achievements**: Unlock rewards as you build productive habits

## Development

This extension uses:
- Chrome Extensions Manifest V3
- Vanilla JavaScript (no frameworks)
- Chrome Storage API for data persistence
- Chrome Notifications API for alerts

## Privacy

This extension respects your privacy:
- All data is stored locally on your device
- No data is sent to external servers
- No tracking or analytics
